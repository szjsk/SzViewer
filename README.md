# SzViewer

### 텍스트뷰어 프로그램 0.0.3

이미지 뷰어/텍스트 뷰어 전환 - ChangeView 버튼 클릭

이미지 뷰어 

`Open` : 파일 열기 (*.jpg *.jpeg *.png *.bmp *.gif)

`size` : 이미지 사이즈 변경 결정 (원본,가로,세로,윈도우 맞춤)

`+` : 이미지 확대 `-` : 이미지 축소

`<<<` : 이전폴더 `>>>` : 다음폴더

`<<` : 첫번째 이미지 `>>` : 마지막 이미지`       

`<` : 이전 이미지 `>` : 다음 이미지
`Split` : 2장보기/1장보기 전환

`ChangeView` : 텍스트뷰어/이미지뷰어 전환

**이미지 뷰어 단축키**  
- ←, → : 이전/다음 파일  
- PgUp, PgDn : 이전/다음 폴더  
- 1 : 윈도우 맞춤  
- 2 : 원본 크기  
- 3 : 가로 맞춤  
- 4 : 세로 맞춤  
- 5, + : 확대  
- 6, - : 축소  
- *.jpg, *.jpeg, *.png, *.bmp, *.gif 만 지원합니다.  
- *webp*는 지원 예정입니다.

텍스트 뷰어

`Open` : 파일열기 (드래그 앤 드롭 가능)	

`Setting` : 폰트설정, 배경색 글자색 설정등

`Search` : 검색

`Split` : 두장보기, 한장보기

**텍스트 뷰어 단축키**  
- ←, → : 페이지 좌우 이동  
- PgUp, PgDn : 폴더 내 다음 파일, 이전 파일  
- 현재 *.txt (UTF-8)만 지원합니다.
