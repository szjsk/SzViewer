# SzViewer

### 텍스트뷰어 프로그램 0.0.1

미완성 베타버전 입니다.

File : 파일열기 (드래그 앤 드롭 가능)	

Setting : 폰트설정, 배경색 글자색 설정등

Search : 검색

Split : 두장보기, 한장보기
